## Game Instructions

### Basic Rules
- You control Team 1 (batting) using the Hit button.
- The computer controls Team 2 (bowling) using the Bowl button.

### Controls
- Hit: Generates random runs (1–6) and animates the ball.
- Bowl: 30% chance of wicket, otherwise Team 2 scores 1 run.
- Add Wicket: Manually increase wicket count.
- Add Over: Manually increase over count.

### Game Over Conditions
The game ends when:
- 5 overs are completed, or
- 10 wickets fall.

A fade-in "Game Over" screen will appear.
